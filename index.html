usuario administrador

  • Utilizar 2FA (Two Factor Authentication)
  • Evitar utilizar misma combinación de usuario y clave en diferentes sitios

Base de datos


  • Ser cuidadosos con usuario y clave de la base de datos
  • No utilizar prefijo wp_ por defecto
  • Hacer backups independientes
  • Hacer backups de posteos en otro formato como xml

Permisos


  • Siempre evitar 777 (rwx rwx rwx)
  • Carpetas por defecto 755 (rwx r-x r-x)
  • Permisos de Archivos 644 (rw- r-- r--)
  • Archivos editables desde escritorio 666 (rw- rw- rw-)


Referencia: https://codex.wordpress.org/es:Modificar_los_permisos_de_ficheros
